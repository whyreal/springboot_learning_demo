# 项目目标

收集了一些 springboot API 项目中的最佳实践，包括：

- 参数获取
- 参数校验
- 统一 response 格式
- 自定义错误，及统一错误处理
- 集成测试的书写
- ...

# springboot 项目初始化

spring 官方提供了一个[网站](https://start.spring.io)，专门用于springboot项目的创建。

本 demo 选择的依赖包括：

- web, Servlet web application with Spring MVC and Tomcat
- jpa, spring data & hibernate
- lombok, 减少模版代码
- devtools， springboot 开发工具
- validating
